Retail Sales & Profitability Analysis (Tableau Dashboard)
Project Overview

This project analyzes retail sales performance using the Superstore dataset.
The objective is to identify patterns in sales, profitability, discounts, and regional performance to support better business decisions.

An interactive Tableau dashboard was developed to explore product performance, customer segments, geographical trends, and the impact of discounts on profitability.

Business Problem

Retail companies often struggle to understand:

Which products generate the most revenue
Which regions contribute most to profit
How discount strategies impact profitability
Which locations or products are loss-making

This project aims to provide insights that help improve sales strategy and profit optimization.

Dataset

Dataset used: Sample Superstore

The dataset contains transactional retail data including:

Order Date
Sales
Profit
Discount
Quantity
Category
Sub-Category
Region
State
Customer Segment

The dataset contains thousands of sales records across different regions in the United States.
